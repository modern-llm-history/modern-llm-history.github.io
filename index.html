GPT以後のLLMの歴史

自己教師型の大規模なベースモデルを前提にしたLLMの歴史は、まだ10年にも満たないごく新しいものです。 2018年の初代GPT登場から現在までの流れを、いくつかの転換点に分けて振り返ります。

🚧 仮設ページ — コンテンツは順次追加されます

ここで扱うのは、とくに「自己教師あり事前学習された大規模ベースモデル」がどのように生まれ、 どんな進化を辿ってきたのかという大まかな物語です。

このページについて

この分野はあまりに流れが速く、「体系的なLLM史」のようなものがきちんと整理されていることは多くありません。 このオンラインブースでは、そんな短くも濃いLLMの歴史を、いくつかの転換点に分けてざっくり振り返っていきます。

2018-01 – 2018-12

GPT-1 と「事前学習する言語モデル」の年

2018年は、いわゆる「GPT以前／以後」の境目になった年です。 OpenAIが "Improving Language Understanding by Generative Pre-Training" というTech Reportを公開し、 のちにGPT-1と呼ばれる最初のモデルが登場します。

モデル自体は、BooksCorpusを使って事前学習された12層のデコーダ型Transformerで、 やっていることは「左から右に次のトークンを予測する」だけの素朴な言語モデルですが、 これをそのまま多様なタスク向けにファインチューニングしてみると、 個別タスク向けに作り込まれたモデルをしのぐ性能を発揮しました。

同じ2018年の後半には、GoogleからBERTが登場します。 こうして2018年は、「巨大な自己教師あり事前学習」が 一部の研究トピックから、NLP全体の前提条件へと変わっていく転換点になりました。

2019-02 – 2020-06

GPT-2 / GPT-3 と「スケーリングがほんとっぽい」時期

2019年2月、OpenAIはGPT-2を発表します。 パラメータ数1.5Bのデコーダ型Transformerで、モデルサイズを増やしていくと、 言語モデリングのlossや各種ベンチマークの性能がきれいなべき乗則に沿って改善していくことが示されました。

2020年夏、GPT-3 175Bが登場します。 プロンプトの先頭に数例の例示を並べるだけで、 翻訳・要約・質問応答・常識推論などを「その場で」こなす few-shot / zero-shot 能力が広く知られるようになりました。

「タスクごとの専用アーキテクチャをいじる前に、 まずはモデルとデータとcomputeを素直にスケールさせるだけでだいぶ強くなる」 という手応えが広く共有された時期です。

2020-06 – 2022-10

汎用ベンチマークと API 化

「何でもそこそこ解ける」汎用モデルが API として提供されるようになり、 産業界での活用が始まります。 主に曖昧な自然言語の分類器としての価値や、チャットボット、インタラクティブなゲームなど、 間違いが許されるエンタメ領域での利用が広がりました。

2022年にはChinchilla論文により「モデルとデータを同じ比率でスケールさせるとcompute最適になる」 という明確な指針が示されます。 Gopherや初期のPaLMなど、大規模モデルの系譜はこの方向を精緻化していきます。

2022-11 – 2024-08

ChatGPTとGPT-4と言語モデル戦国時代

ChatGPTというあまりにも大きな起爆剤が登場し、LLMの存在が一般社会に広く知れ渡ります。 そしてGPT-4の衝撃。 2024年にはClaude 3 Opus、Gemini 1.5 Pro、GPT-4oなど、各社のモデルが次々とリリースされ、 言語モデル戦国時代と呼ぶにふさわしい競争が始まりました。

オープンソース側でもLLaMAから始まったローカルLLMブーム、 Mistral、Gemmaの台頭と、モデルの民主化が進みます。 GitHub CopilotやClineなど、コーディング支援ツールも登場し始めます。

2024-09 – 2025-08

Reasoner: What Ilya saw

ついにReasonerが到来します。 事前学習のスケーリングに続く第二のスケーリング — Test-Time Scaling。 o1, o3、そしてClaude 3.5 Sonnet の改良版がこの潮流を牽引しました。

2025年に入り、モデルリリースのペースはさらに加速。 Claude 4, Gemini 2.5, GPT-5とメジャーリリースが続き、 IMO金メダル相当の数学問題解決やハルシネーション率の大幅低減など、 質的な飛躍も見られました。

2025-09 – 2026-03

加速し続けるLLM

2025年はエージェント元年でした。 LLMファーストのコーディング、研究分野での活用、 人間の手を介さないオフィスワークが現実味を帯び始めます。

GPT-5.1, Claude Opus 4.5, Gemini 3.0 Proなどがリリースされ、 モデルリリース間隔はますます縮み始めます。 LLMがLLM自体の技術を加速する瀬戸際 — それが2026年の今です。
